# majority_game

⬡ Majority — стратегическая настольная игра на захват клеток. Концептуальный прототип.  
В репозитории есть одиночный прототип в файле `majority_game.html` и LAN-лобби на Flask для совместной игры (до 5 игроков).

## Быстрый старт (LAN)

1) Установите зависимости:

```bash
python -m venv .venv
# Windows:
.venv\\Scripts\\activate
# Linux/macOS:
# source .venv/bin/activate

pip install -r requirements.txt
```

2) Запустите сервер на ПК-хосте:

```bash
python -m server.app
```

3) Откройте в браузере:
- На хосте: `http://127.0.0.1:5000`
- На другом ПК в той же сети: `http://<IP_хоста>:5000`

## Что уже сделано
- Ввод никнейма, выбор формата.
- Создание лобби и ожидание соперников.
- Присоединение по коду или из списка открытых лобби.
- До 5 игроков в одном лобби.
